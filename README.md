# BDevC Work App Help

This is just the name I could come up with, lol.
This website was developed with React js + TypeScript SWC + Tailwind css

This website is meant to help the working class or anybody to create their own Resume / Curriculum Vitae which would be automatically organised and structured for them.

The designs are functioning effectively and are responsive, so you can use it with your smart phones, tablets, laptops, bigger screens etc or any device that can access the internet.

You can copy or download your resume / CV when you are done, your choice.

This website is customize so that any changes you make it immediately reflects, just like typing in microsoft word only difference is that you don't need to press enter to move to the next line or other things because this website is designed to automatically organise your work for you when you are done without needing to do them yourself.

This is a temporal appearance as new improvements and updates would be done from time to time.
